1. открыть в PyCharm заранее созданную папку
2. внутри терминала PyCharm создать виртуальное окружение:
python -m venv venv

3. установить в качестве интерпретатора интерпретатор venv:
File->Settings-> найти текущий проект и добавить для него
интерпретатор python.exe по пути venv\Scripts

4. поставить зависимости в окружение:
.\venv\Scripts\activate (активация окружения)
pip install Django, python-dotenv, psycopg2

5. запустить старт проекта. должна появиться соотв. директория: 
django-admin startproject <имя>

6. перейти в директорию проекта создать приложение:
python manage.py startapp <имя>

7. в директории проекта теперь можно запускать приложение:
python manage.py runserver
